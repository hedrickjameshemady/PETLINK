# PETLINK: Pet Management, Adoption System, and Lost Found

This system is designed to manage pet shelter operations and simplify the adoption process. It allows users to view available pets, submit adoption requests, and helps administrators manage records efficiently.

## Tech Stack

* Frontend: React.js
* Backend: Java / C#
* Database: MySQL
* Tools: GitHub, VS Code
